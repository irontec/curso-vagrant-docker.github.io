## Contenedores Docker

> Un contenedor es una instancia en tiempo de ejecución de una imagen de docker.

> Un contenedor de docker consiste en:
>    - Una imagen de docker
>    - Un entorno de ejecución
>    - Un conjunto estándar de instrucciones
